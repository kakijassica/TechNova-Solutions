# 🚀 TechNova Solutions

## DevOps Project 1 - Static Website Deployment using AWS EC2, Nginx & Jenkins

TechNova Solutions is a responsive multi-page static website developed using HTML, CSS, and JavaScript. This project is created as the foundation for learning and implementing DevOps concepts.

The project will be deployed on an AWS EC2 instance using the Nginx web server and automated using Jenkins CI/CD.

---

## 📌 Project Overview

This project demonstrates:

- Responsive Frontend Development
- Git & GitHub Version Control
- Static Website Deployment
- AWS EC2 Hosting
- Nginx Web Server Configuration
- Jenkins CI/CD Automation
